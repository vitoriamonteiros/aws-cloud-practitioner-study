# AWS Pricing Calculator

É a ferramenta para **estimar os custos futuros** de serviços AWS com base em configurações e uso planejado. <br>
Permite criar cenários personalizados e obter uma previsão detalhada dos custos antes de iniciar a migração ou implementação. (projeção).
